# Programming 2 - Machine Problem: Executive Sales Summary Dashboard
**University of Perpetual Help System DALTA – Molino Campus**
**Author:** Travis Peñalosa

## Overview
This is a Java Swing-based Graphical User Interface (GUI) application designed to parse, validate, and analyze the `vgchartz-2024.csv` dataset. It acts as an Executive Sales Summary Dashboard, computing global revenue, averages, and highest/lowest performing video game titles.
